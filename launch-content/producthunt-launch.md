## Maker Comment (Response to Common Questions)

**Q: How is this different from AutoGPT or GPT-Engineer?**  
Great question! AutoGPT is fully autonomous (you give it a goal, it figures out the steps). AgentForge is *semi-autonomous* â€” you design the workflow, the agent executes it. This gives you control over what the agent can/can't do. Think of it like the difference between hiring a freelancer (AutoGPT) vs. building an internal tool (AgentForge).

**Q: Can I connect my own tools/APIs?**  
Yes! You can add custom API nodes, webhook nodes, database queries, web scraping, and even custom Python functions. If you can call it from code, you can add it to AgentForge.

**Q: What LLMs are supported?**  
GPT-4, GPT-4-turbo, Claude Sonnet 4.5, Claude Opus, DeepSeek, Gemini Pro. You can BYOK (bring your own keys) or use ours with pay-as-you-go pricing.

**Q: What happens if an agent fails mid-execution?**  
We have automatic retry logic with exponential backoff. You can also set up fallback nodes (if this fails, do that). All execution logs are saved, so you can debug exactly where it broke.

**Q: Is there a self-hosting option?**  
Not yet, but it's on the roadmap! If you need it, let me know â€” we're prioritizing features based on demand.
