# Weather Forecast

Get current weather and forecasts for any city using the free Open-Meteo API (no API key required).

## Usage
```bash
./weather.sh "Oslo"
./weather.sh "New York"
```

## Output
Returns current temperature, wind speed, and weather condition.

## Safety
- Read-only HTTP requests to open API
- No API key or authentication needed
- No data stored or transmitted beyond the query
