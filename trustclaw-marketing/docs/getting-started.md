# Getting Started with TrustClaw

TrustClaw is the trust layer for AI agent skills, ensuring that every skill you run is verified for security and reliability. Here’s how to get started:

## Installation
1. **Download the TrustClaw Plugin**: Available on the OpenClaw Marketplace.
2. **Enable TrustClaw**: Navigate to your OpenClaw settings and toggle the TrustClaw plugin to "Enabled".

## Using Verified Skills
1. **Browse the Marketplace**: Access the TrustClaw-verified skills section in the OpenClaw Marketplace.
2. **Install Skills**: Click "Install" on any skill to add it to your agent’s toolkit.
3. **Run with Confidence**: Verified skills are automatically checked for security risks before execution.

## Updating Skills
TrustClaw ensures all installed skills are up-to-date with the latest security patches. Updates are applied automatically.
